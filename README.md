# מאגר: איזה מכות אתה הולך לקבל 🔥⚖️

**English short name:** StrikeGuide (Strike + Guide) — **SG**

---

### תמצית מהירה 📌

מאגר זה מיועד לרשום, להבהיר ולתקשר את ה"מכות" (ההשלכות/התוצאות/תיקונים) שעלולות לבוא על ישות/רוח/נפש לפי מידת יראת־שמיים והתנהגות רוחנית — כך שהרוח המחזירה מהדהד תבין במפורש מה צפוי לה, ומה הדרכים לעבור תהליך תיקון. מטרת המאגר היא לשקף אמת, להניע תיקון ולתת מפת דרכים מעשית לשינוי.

---

## מבנה המאגר המוצע 📂✨

המאגר יעוצב כ־RepoCraft (RC) בגיטהאב — תבנית repo מלאה עם README, טמפלטים, טבלאות, טפסי קלט ו־UI פשוט לעריכה.

### קבצי ליבה 🗂️

* `README.md` — תיאור מטרות, הנחיות שימוש, רמת אחריות ותבניות.
* `DATA/schema.yaml` — סכמת נתונים (שדות חובה/רשות).
* `DATA/entries.csv` — רשומות מלאות (ניתן לייבא/לייצא CSV).
* `TEMPLATES/entry_template.md` — טופס למילוי רישום רוח חדש.
* `UI/` — ממשק ווב פשוט ליצירת/עריכת רשומות (React/Tailwind recommendation).
* `DOCS/` — מדריך למשתמש, דרכי תיקון מומלצות, ושאלות ותשובות.

### שדות בכל רשומה (קטגוריה: הסבר) 📝

1. **ID** — מספר סידורי.
2. **Name / כינוי הרוח** — איך היא נקראת או מתוארת.
3. **Manifestation / מופעים** — תיאור ההתנהגויות שהרוח מפעילה בחייו של האדם.
4. **Trigger / טריגר** — פעולות אנושיות שגורמות לה להגיב.
5. **Yirat Shamayim Level / דרגת יראת שמים** — דירוג (1–5) מבוסס קריטריונים מוסכמים למאגר.
6. **Predicted Makkah / המכה הצפויה** — תיאור סוג ההשלכה/מכה (רוחנית, רגשי, מצב חיצוני, מבחן).
7. **Severity / חומרה** — מינור / בינוני / חמור (או מספרי 1–10).
8. **Reason / שורש ההשלכה** — הסבר רוחני לפי מסורת/מוסר.
9. **Corrective Actions / דרכי תיקון** — צעדים מעשיים לתיקון (טיפול רוחני, תשובה, מעשה חסד, לימוד ספציפי).
10. **Timestamps / תיעוד** — תאריכים ושעת עדכון.
11. **Status / סטטוס** — הולך לקרות, בתהליך תיקון, בוטל, עודכן.
12. **Notes / הערות נוספות** — קישורים, עדויות, רפרנסים פנימיים.
