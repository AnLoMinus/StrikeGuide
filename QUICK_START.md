# התחלה מהירה / Quick Start

## 🚀 התקנה מהירה ב-3 שלבים / Quick Installation in 3 Steps

### 1️⃣ הורדה / Download
הורד את הקובץ **`StrikeGuide-complete.zip`** וחלץ אותו.

### 2️⃣ התקנה / Installation
```bash
chmod +x install.sh
./install.sh
```

### 3️⃣ הפעלה / Run
```bash
cd UI
npm run dev
```

פתח את הדפדפן בכתובת: `http://localhost:3000`
