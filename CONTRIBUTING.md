## הנחיות ספציפיות ל-StrikeGuide / StrikeGuide Specific Guidelines

### מטרת התרומות / Purpose of Contributions

תרומות למאגר זה צריכות להיות:
- ✅ **מטרות תיקון** - מסייעות לתיקון רוחני
- ✅ **מדויקות** - מבוססות על אמת ומסורת
- ✅ **מכבדות** - שומרות על כבוד האדם
- ✅ **אתיות** - עומדות בעקרונות ETHICS.md ו-CONDUCT.md

### סוגי תרומות / Types of Contributions

#### 1. תרומות תוכן / Content Contributions
- הוספת רשומות חדשות (לפי התבנית)
- שיפור דרכי תיקון
- הוספת מקורות ולימודים

#### 2. תרומות טכניות / Technical Contributions
- שיפור הממשק
- תיקון באגים
- שיפור ביצועים
- תיעוד

#### 3. תרומות תיעוד / Documentation Contributions
- שיפור README
- הוספת דוגמאות
- תרגום

### תהליך תרומה / Contribution Process

#### שלב 1: הכנה / Step 1: Preparation
1. קרא את `README.md`
2. קרא את `CONDUCT.md` ו-`ETHICS.md`
3. ודא שהמאגר הוא public ו-maintable

#### שלב 2: יצירת תרומה / Step 2: Creating Contribution
1. Fork את המאגר
2. צור branch חדש: `git checkout -b feature/your-feature-name`
3. בצע את השינויים
4. ודא שהכל עובד

#### שלב 3: בדיקה / Step 3: Review
1. ודא שהקוד עובד
2. ודא שהתוכן עומד בעקרונות
3. ודא שאין פרטים מזהים

#### שלב 4: הגשת תרומה / Step 4: Submitting
1. Commit את השינויים
2. Push ל-branch שלך
3. פתח Pull Request
4. הסבר את התרומה

### כללי תרומה / Contribution Rules

#### ✅ מותר / Allowed
- הוספת רשומות חדשות לפי התבנית
- שיפור דרכי תיקון
- תיקון שגיאות
- שיפור ממשק
- תיעוד

#### ❌ אסור / Forbidden
- הוספת פרטים מזהים
- שימוש לרעה
- השמצה או ביקורת שלילית
- תוכן לא אתי
- שינויים שמפרים את העקרונות

### בדיקת תרומות / Review Process

כל תרומה תיבדק לפי:
1. **עמידה בעקרונות** - CONDUCT.md, ETHICS.md
2. **איכות** - תוכן מדויק ומדויק
3. **כוונה** - מטרה תיקונית
4. **טכנית** - קוד נקי ופועל
